# Python para Trading Algorítmico Avanzado — Parte 1: Estrategias de Aperturas y Gaps

Repositorio de código completo del **Libro 1** de la serie "Python para Trading Algorítmico Avanzado" de Tirso Díaz Díaz.

## 📖 Sobre este libro

Este repositorio contiene todo el código Python funcional del **Libro 1**, centrado en estrategias de aperturas de mercado y gaps. Aprenderás a explotar las ineficiencias estadísticas que ocurren en momentos específicos del día mediante:

- Estrategias de Ruptura del Rango de Apertura (ORB)
- Estrategias de Desvanecimiento de Gaps (Gap Fade)
- Análisis de gaps inter-sesión (Sídney-Tokio-Londres-Nueva York)
- Clasificación y medición estadística de gaps
